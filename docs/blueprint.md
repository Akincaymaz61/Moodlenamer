# **App Name**: Groove Illusion

## Core Features:

- Song Generation: Generates fake, but realistic-sounding, song titles and artist names using an AI tool.
- Track Listing: Displays the generated song titles and artist names in a scrollable list format.
- Refresh Playlist: Allows users to refresh the song list with a new set of generated songs.
- Simulated Playback: A 'play' button that, when activated, provides a visual indication of playback without any actual audio.
- Title Search: Enables users to search for song titles within the generated list.

## Style Guidelines:

- Background color: Dark charcoal (#222222) to establish a modern dark theme.
- Primary color: Vibrant purple (#A020F0) for active elements, providing a bold accent.
- Accent color: Electric indigo (#6F00FF) to highlight key interactive elements.
- Body and headline font: 'Inter', a grotesque-style sans-serif, for a modern and neutral look suitable for both headlines and body text.
- Use minimalist, geometric icons in the primary purple color to represent actions and song details.
- Modern, edge-to-edge layout with focus on clean typography and vibrant accents.
- Subtle transitions and animations for playlist refreshes and playback indicators, enhancing the user experience.
